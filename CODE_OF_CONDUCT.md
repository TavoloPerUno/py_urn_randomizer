# Contributor Covenant Code of Conduct

This project has adopted the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its Code of Conduct.

Please read the full text at the link above. By participating in this project
you agree to abide by its terms.

## Reporting

Instances of unacceptable behavior may be reported by opening an issue or
contacting the project maintainers.
